# 🚀 Team Task Management System
The Team Task Management System is a simple yet powerful tool that helps teams handle projects more effectively by organizing tasks, assigning them to the right members, and tracking progress in real time. Its focus on clarity and accountability ensures smoother collaboration, timely completion of work,improved overall efficiency.


A full-stack MERN application to manage projects, assign tasks, and track progress with role-based access.

---

## 🔥 Features

- User Authentication (Login/Register)
- Role-based Access (Admin / Member)
- Task Assignment
- Task Status Tracking
- Dashboard Overview

---

## 🛠 Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Deployment: Render & Vercel

---

## 👤 Roles
- Admin
- Members
### Admin
- Create tasks
- Assign tasks

### Member
- View tasks
- Update assigned tasks
